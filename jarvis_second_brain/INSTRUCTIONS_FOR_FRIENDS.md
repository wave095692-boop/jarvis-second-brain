# 🤖 คู่มือติดตั้งระบบควบคุม Jarvis สำหรับเพื่อน (Jarvis Second Brain Control Node)

คู่มือนี้จะช่วยให้คุณเชื่อมต่อเครื่อง Mac ของคุณเข้ากับระบบควบคุมของบอส เพื่อให้คุณสามารถรันและสั่งการเว็บต่าง ๆ (เช่น เปิดแอป YouTube บน Brave Browser) ได้โดยตรงจากหน้าจอคลาวด์/รีโมท

---

## 📋 สิ่งที่ต้องมีก่อนเริ่ม (Prerequisites)
1. **เครื่อง macOS (Mac)**
2. **มี Python 3** ติดตั้งอยู่ในเครื่อง (สำหรับรันตัวคอยควบคุมหลังบ้าน)
3. **ติดตั้ง Brave Browser** ไว้ที่โฟลเดอร์แอปพลิเคชันปกติ (`/Applications/Brave Browser.app`)

---

## 🚀 ขั้นตอนการติดตั้งและรันระบบ

### 1. ดาวน์โหลดโฟลเดอร์โปรเจกต์
ดาวน์โหลดโฟลเดอร์ `jarvis_second_brain` หรือดาวน์โหลดโค้ดผ่านคำสั่ง Git:
```bash
git clone https://github.com/wave095692-boop/jarvis-second-brain.git
cd jarvis-second-brain/jarvis_second_brain
```

### 2. สตาร์ตระบบควบคุมหลังบ้าน (Local Server)
เปิดโปรแกรม **Terminal** บน Mac ของคุณ เข้าไปยังโฟลเดอร์โปรเจกต์นี้ จากนั้นพิมพ์คำสั่ง:
```bash
bash run_dashboard.sh
```
*ระบบจะทำการปิดพอร์ตเก่าที่ค้างอยู่ สตาร์ตเซิร์ฟเวอร์ควบคุมเบื้องหลังบนพอร์ต `8500` และเริ่มเก็บประวัติบันทึกการทำงาน*

เมื่อรันสำเร็จ คุณจะเห็นข้อความดังนี้:
```text
--------------------------------------------------------
🤖 JARVIS SECOND BRAIN DASHBOARD LAUNCHED
URL: http://localhost:8500
LAN: http://192.168.1.XX:8500
--------------------------------------------------------
```

---

## 🎮 วิธีการใช้งาน
เมื่อคุณรันเซิร์ฟเวอร์โลคอลในเครื่องแล้ว คุณสามารถควบคุมและสั่งการได้สองแบบ:
1. **รันผ่านเว็บโลคอล:** เปิดเบราว์เซอร์แล้วเข้าไปที่ `http://localhost:8500`
2. **รันผ่านหน้าเว็บคลาวด์:** เมื่อบอสแชร์หน้าเว็บหลัก (เช่น จากลิงก์คลาวด์ของ Render) หน้าเว็บคลาวด์จะฉลาดพอที่จะส่งสัญญาณควบคุมกลับมายังพอร์ต `8500` ในเครื่องของคุณโดยตรง

### 🎥 การทำงานกับ Brave Browser
- เมื่อคุณกดปุ่ม **Launch YouTube on Brave** หรือ **Open YouTube** บนหน้าควบคุม:
  1. ระบบจะสั่งเปิดแอปพลิเคชัน **Brave Browser** ของคุณ
  2. เปิดหน้าเว็บ `https://www.youtube.com/` บน Brave โดยอัตโนมัติ

---

## 🛠️ การแก้ไขปัญหาเบื้องต้น
- **กรณีขึ้นข้อความสีแดง DISCONNECTED หรือรันคำสั่งไม่ได้:**
  - ให้ตรวจสอบว่าคุณได้พิมพ์ `bash run_dashboard.sh` แล้วหรือยัง
  - ตรวจสอบว่ามีโปรแกรมอื่นใช้งานพอร์ต `8500` อยู่หรือไม่ (ตัวรันจะพยายามเคลียร์พอร์ตให้โดยอัตโนมัติ)
- **กรณีระบบไม่ยอมเปิด Brave Browser:**
  - ตรวจสอบว่ามี Brave Browser อยู่ในเครื่องและติดตั้งไว้ในโฟลเดอร์ `/Applications/` (ไม่ใช่โฟลเดอร์อื่น)
